What is Superdog Event?

The Superdog Event is an online storaged page is used store data from a "superdog event". "Superdog event" is a Canadian dog's event where all the dog's lover gather together. The mission of this website is collecting data on every event for the operator, from that, they can see how fast and big they've grown.

Technologies Used: HTML, JavaScript, CSS, Bootstrap.
